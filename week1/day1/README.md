Day 1 – SQL Practice
🔗 Overview

This repository contains my Day 1 SQL practice work.

The purpose of this session was to understand the fundamentals of SQL and build confidence in writing queries that retrieve, filter, sort, group, and analyze data from a database.

Learning Objectives

During this practice session, I focused on:

Understanding the structure of SQL queries
Retrieving data from database tables
Applying conditions using WHERE clause
Using pattern matching with LIKE
Working with date functions
Using aggregate functions such as SUM(), AVG(), COUNT()
Grouping records using GROUP BY
Filtering grouped data using HAVING
Sorting records using ORDER BY
Performing table joins using JOIN operations
Writing subqueries for advanced filtering

Topics Covered:

1.Basic Queries:
Select all columns from a table
Select specific columns
Filter records using conditions
Search using wildcard characters

2.Date Queries:
Employees hired in specific years
Employees hired before or after certain dates
Employees hired in the last 2 years

3.Aggregate Queries:
Total salary calculation
Average salary calculation
Minimum and maximum salary
Employee count by department
Group By & Having Queries
Department-wise salary analysis
Filtering grouped records
Counting employees department-wise

4.Order By Queries:
Sorting employees by salary
Sorting employees by age
Sorting using multiple columns

5.join Queries:
Employee and department details
Employee and project details
LEFT JOIN operations
Subqueries with departments
